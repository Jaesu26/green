Variable Name

  • 변수 줄임말
  • 변수 줄임말은 아니지만 아무튼 줄임말임
  • 변수 이름 짓기

변수 줄임말

- 변수를 줄임말로 쓸 때가 있는데 뭐인지 헷갈리는 변수를 정리할 거임

- 나중에 변수 줄임말 말고도 변수명을 어떻게 정할지에 대한 내용도 정리할 수 도 있음

- rtn ---> return

- tmp ---> temporary

- num ---> number

- lib ---> library

- lin ---> linear

- rv ---> random variable

- aes ---> aesthetic

- axes ---> axis(축)의 복수형

- param(s) ---> parameter(s)

- loc ---> location

- pch ---> point character

- fig ---> figure

- ax ---> axes

- alg ---> algebra(대수학)

- cnt ---> count

- ec ---> edgecolor

- fc ---> facecolor

- h ---> horizontal

- v ---> vertical

- cpy ---> copy

- tb ---> table

- autopct ---> autopercentage

- ptr ---> pointer

변수 줄임말은 아니지만 아무튼 줄임말임

- WLLN ---> Weak Law of Large Numbers(큰 수의 약한 법칙)

- MLE ---> maximum likelihood estimate(최대우도추정량)

변수 이름 짓기

- 모음을 생략하여 짓기 count ---> cnt, return ---> rtn

- 임시 변수는 _ 붙이기 ---> tmp_

- 클래스 내부 객체이름 앞에 _를 붙이면 인스턴스 객체에 .을 붙이고 tab을 눌러서 확인하지 못함(dir 함수를 써야한다)
